Class GeoNamesResultParserMethods
 
Packageorg.geonames
Classpublic class GeoNamesResultParser

Class that contains internal helper functions used to parse the XML returned from the GeoNames servers to create the relevant objects. Only meant as helpers for the library; NOT meant for use by developers using the library.
